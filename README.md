## What's HOTATE?

「HOTATE」は、ユーザに対して本との出逢いを増やして視野を広く持ってもらいたいという思いから立ち上がったサービスです。ユーザは本の検索やタグの管理、その他様々な便利な機能を用いて本を楽しむことが出来ます。

想定している場所は図書館などの施設ですが、もちろん個人が運用することも可能です！
